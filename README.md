# IT理解度確認テスト（quiz-repo）

このリポジトリは、初心者向けの基礎級から、最高難易度である「小学生向けIT理解度確認テスト」までを網羅したWebクイズアプリケーションです。
HTMLのみのシンプルな構成で、ブラウザ上ですぐにクイズに挑戦できます。

## 🌐 公開サイト
テストは以下のURLから実際に受検することができます。
👉 **[IT理解度確認テスト 公開サイト](https://jr-it.rweb.site/)**

## 📂 テストの構成（難易度・位置づけ）

本リポジトリは、各レベルごとにファイルが分かれています。

*   **🏆 小学生向けit理解度確認テスト**：本シリーズにおける**最高峰・最難関**の理解度テスト。高度なITリテラシーと深い知識が求められます。
*   **🥇 1級**：最難関への登竜門となる、非常に高度なIT知識を試すクラス。
*   **🥈 準1級**：応用的なIT知識や、トレンド技術に関するレベル。
*   **🥉 2級**：実務や日常で役立つ、やや踏み込んだITリテラシー問題。
*   **🔰 3級**：ITの基本的な知識・用語を問うエントリークラス。
*   `index.html`：各テストへのリンクをまとめたトップページ。

## 🛠️ テクノロジー
*   **HTML5 / JavaScript**
*   ※ サーバーサイドを必要とせず、静的ホスティング（GitHub Pagesなど）でそのまま軽量に動作します。

## 🚀 ローカルでの実行方法

1.  このリポジトリをクローンまたはダウンロードします。
   git clone https://github.com/jr-it-ex/quiz-repo.git
    
2.  クローンしたフォルダ内にある `index.html` を、お使いのブラウザ（Google Chrome, Edge, Safariなど）にドラッグ＆ドロップするだけで、ローカル環境でテストを実行できます。

# IT Literacy Test (quiz-repo)

This repository is a web-based quiz application that covers everything from beginner-level questions to the highest difficulty level: the “IT Literacy Test for Elementary School Students.”
With a simple HTML-only structure, you can take the quiz right away in your browser.

## 🌐 Public Site
You can take the tests at the following URL.
👉 **[IT Proficiency Test Public Site](https://jr-it.rweb.site/)**

## 📂 Test Structure (Difficulty Levels and Classification)

This repository is organized with separate files for each level.

*   **🏆 IT Proficiency Test for Elementary School Students**: The **most advanced and challenging** proficiency test in this series. It requires advanced IT literacy and in-depth knowledge.
*   **🥇 Level 1**: A class that tests extremely advanced IT knowledge, serving as the gateway to the most challenging level.
*   **🥈 Pre-Level 1**: A level covering applied IT knowledge and trending technologies.
*   **🥉 Level 2**: Questions on slightly more in-depth IT literacy that are useful in practical work and daily life.
*   **🔰 Level 3**: An entry-level class testing basic IT knowledge and terminology.
*   `index.html`: The homepage that compiles links to each test.

## 🛠️ Technology
*   **HTML5 / JavaScript**
*   *Note: No server-side processing is required; it runs lightweight as-is via static hosting (such as GitHub Pages).

## 🚀 How to Run Locally

1.  Clone or download this repository.
   git clone https://github.com/jr-it-ex/quiz-repo.git
    
2.  Simply drag and drop the `Level_to_Take/index.html` file from the cloned folder into your browser (Google Chrome, Edge, Safari, etc.) to run the test in a local environment.
